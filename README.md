# mgs

tiny database for mgs

# usage

mgs.py ==> show all girl's videos

mgs.py girlname ==> show all videos about this girl

mgs.py girlname video ==> use vlc to play the video

# /tmp/xxoo struct like below

```shell
/tmp/xxoo/
├── name1
│   ├── name1.mp4
│   ├── new_video
│   │   └── name1_new.mp4
│   ├── tmpfile.txt
│   └── vd.m4v
├── name2
│   ├── name2.mp4
│   └── vimgrep.m4v
├── name3
│   ├── video3.mp4
│   └── video.wmv
├── tmpfile.txt
├── video4.mp4
└── video.wmv
```
